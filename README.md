# 🍔 Mr. Donalds

Uma releitura moderna da experiência do McDonald's, desenvolvida com foco em performance, design e boas práticas de desenvolvimento full stack.

---

## 🚀 Sobre o projeto

O **Mr. Donalds** é uma aplicação web que simula a interface e funcionalidades de uma plataforma de pedidos de fast food, trazendo uma abordagem moderna utilizando tecnologias atuais do ecossistema JavaScript.

O projeto foi desenvolvido como forma de prática e evolução em desenvolvimento full stack, com foco em:

- Arquitetura escalável
- Integração com banco de dados
- Autenticação de usuários
- Interface moderna e responsiva

---

## 🛠️ Tecnologias utilizadas

- ⚡ Next.js — Framework React para aplicações modernas
- 🟦 TypeScript — Tipagem estática para maior segurança
- 🧠 Prisma ORM — Manipulação de banco de dados de forma eficiente
- 🐘 PostgreSQL — Banco de dados relacional
- ☁️ Neon DB — Banco de dados PostgreSQL serverless
- 🎨 Tailwind CSS — Estilização rápida e responsiva
- 🎬 Framer Motion — Animações modernas

---

## 🔐 Funcionalidades

- ✅ Sistema de autenticação de usuários
- ✅ Integração com banco de dados (Neon DB)
- ✅ Listagem de produtos/restaurantes
- ✅ Interface inspirada em grandes plataformas
- 🚧 Funcionalidades em expansão

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/Projeto-mr-donalds.git

Acesse a pasta:

cd Projeto-mr-donalds

Instale as dependências:

pnpm install
⚙️ Configuração do ambiente

Crie um arquivo .env na raiz do projeto:

DATABASE_URL="sua_string_do_neon_db"
🗄️ Banco de dados

Execute as migrations:

pnpm prisma migrate dev

Visualizar banco:

pnpm prisma studio
▶️ Rodando o projeto
pnpm dev

Acesse:

http://localhost:3002
